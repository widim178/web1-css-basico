Aplica las propiedades que queres dar al elemento seleccionado en el Selector.
Propiedades

Valor de Propiedad

1.5- Un selector es lo que permitira aplicar estilos a elementos específicos de HTML. De esta forma es posible ser preciso al momento de aplicar diseño a un sitio web

1.6- Selector

Selector Universal
Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *
Selector por Etiquetas
Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar "<.>". Por ejemplo
Selector decendente
Con este selector seleccionamos un elemento que esta dentro de otro del mismo tipo. Estos seectores siempre estan formados por dos o mas selectores separados entre si. El ultimo selector sera el elemento HTML al que se le aplicara las propiedades de CSS
Selector de clases
Los selectores de clases comienzan siempre por un punto. Los clientes web (navegadores) identifican este punto y lo distinguen del resto de los selectores. Para definir en HTML el uso de los selectores de clase, se utiliza el atributo class.
Las clases pueden ser utilizadas varias veces y en distintas etiquetas, lo que nos hace pensar que es necesario ver cuales son la características que queremos aplicar a las etiquetas que sean comunes a varias

1.7- Procesa el documento en una serie de etapas.

  1. El navegador carga el HTML. pro ejemplo lo recibe la red
  2. Convierte el HTML en un DOM (modelo de objetos del documento). El DOM representa el documento en la memoria del ordenador
  3. El navegador va a buscar representa el documento en la memoria del ordenador, y tambien al CSS vinculado
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas segun el tipo de selector. Para cada tipo de selector que encuentre, calcula que reglas deben aplicarse y a que nodos en el DOM se les aplica el estilo segun corresponda (este paso intermedio se llama árbol de renderizacion)
  5. El arbol de renderizacion presenta la estructura en que los nodos deben aparecer despues de aplicarles las reglas
  6. En la pantalla se muestra el aspecto visual de la página (esta etapa se llama pintura)
